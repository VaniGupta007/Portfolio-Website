Project-3 (Web Development)📆

I recently built my very first web development project: a Social Links Page.

It may seem like a simple project, but for me it was much more than writing HTML and CSS. It was my introduction to turning ideas into something people can actually use and interact with.

What I learned from this project :

• Structuring web pages using HTML
• Styling and designing responsive layouts with CSS
• Understanding how small design choices improve user experience
• Debugging errors
• Realizing that every challenge teaches you something new

Tools I used :

| Visual Studio Code
| HTML
| CSS
| Chrome Developer Tools

The best part wasn't just completing the project. It was discovering how satisfying it feels to build something from scratch. Watching lines of code transform into a working webpage made me even more curious to explore web development. This project reminded me that learning isn't about creating something perfect. It's about creating something, learning from it, and making the next version even better.
